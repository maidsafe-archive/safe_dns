Metadata of Tour is updated in Home. Thus when a Container is created, the function returns the created Container and also the parent of the Container in which the new Container is being returned. Based on the above example, when the Container Tour is created in Pictures, this function would return a tpule of (Tour, Home) In case if there is no parent for the Container then None is returned.
